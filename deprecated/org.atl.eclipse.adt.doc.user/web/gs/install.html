Installation of the ATL plug-in

Our development is made under the Eclipse 3.0M8. Others versions have not been tested but the ATL plug-ins shall run under, at least, an upper version of Eclipse, and propably on every 3.0Mx version.

Installing via the Update Manager

The Eclipse ADT features can be found on the ATL update site at http://www.sciences.univ-nantes.fr/lina/atl/updates/. To locate and install them into a product, you can look at the Eclipse examples installation procedure at Installing the examples.

Note : The update via Help > Software updates > Manage configuration may cause some problems for finding all new features and will show up only the version just following yours. If you do not want to make an update for every newer features, use Help > Software updates > Find and install page that has to be correct.

Installing manually

You should not be able for the moment to install the ATL plug-ins manually, because the zip files are not yet published on the server.

Dependencies

You need to install before the Eclipse EMF plug-in which one you can found on the server or at the Eclipse project URL.
